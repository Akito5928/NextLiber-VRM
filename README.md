# NextLiber VRM (NLV)

**Unity の外側で Unity プロジェクトを動かす。  
NextLiber VRM は、Unity の資産を解放するための  
次世代 Virtual Runtime Machine 構想です。**

---

## 🌐 What is NextLiber VRM?

NextLiber VRM（NLV）は、  
**Unity のランタイムに依存せずに Unity プロジェクトを実行するための  
“外部仮想ランタイム（Virtual Runtime Machine）”を  
Java 上に構築することを目指すプロジェクトです。**

現時点では構想段階であり、  
技術検証と思想の公開を目的としています。

---

## 🔥 Why?

Unity のエディタとアセット構造は強力ですが、  
ランタイムは以下の課題を抱えています。

- 重い  
- 閉じている  
- 企業・教育・研究用途で扱いづらい  
- ライセンスが不安定  
- セキュリティ監査が困難  

**Unity の資産は使いたい。  
しかし Unity のランタイムには縛られたくない。**

NextLiber VRM は、この矛盾を解消するための試みです。

---

## 🚀 Vision

- Unity の Scene / Prefab / C# を **外部ランタイムで実行可能にする**  
- Java の安定性・セキュリティ・企業適性を **Unity の世界に持ち込む**  
- Unity のランタイム依存から **解放（Liberation）** する  
- ゲームエンジンの構造を **再定義（Next）** する  
- Unity の外側に存在する **次世代 Virtual Runtime Machine** を構築する  

**“Unity の外側に Unity を動かす”**  
この思想そのものが NextLiber VRM の核です。
